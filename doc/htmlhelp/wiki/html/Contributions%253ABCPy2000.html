Synopsis

BCPy2000 is a set of modules for BCI2000 that allows a highly customizable experiment design using the Python scripting language. It was developed by Thomas Schreiner at the Max-Planck-Institute for Biological Cybernetics in TÃ¼bingen / Germany. An extensive amount of relevant documentation is contained in the project thesis, which is available for download here.
