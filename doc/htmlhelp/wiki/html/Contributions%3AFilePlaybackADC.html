Synopsis

A source module that replays EEG data from BCI2000 .dat files.

Location

http://www.bci2000.org/svn/trunk/src/contrib/SignalSource/FilePlayback

Versioning

Author

Adam Wilson

Functional Description

The FilePlaybackADC reads data from a prerecorded BCI2000 .dat file. State information is not read from the file but re-created during operation. This may lead to unexpected results.

The FilePlaybackADC allows adjusting the speed of replaying, the location in the file where to start replay, and whether replay should be re-started at the end of the file.

Parameters

SourceCh

number of digitized and stored channels

SampleBlockSize

number of samples transmitted at a time

SamplingRate

sample rate, must match the input file's sampling rate

SourceChList

a list of channels to acquire from the input file (empty for all)

Speedup

a value indicating the amount that the replay should be sped up

FileName

the path to the existing BCI2000 data file

SuspendAtEnd

suspend execution when the end of the data file is reached

StartTime

the temporal offset where replay should begin

States

None.
