How to get Matlab support working from Borland C++ Builder:

1) copy the following Matlab library files into the program directory:
   \matlabr12\extern\lib\win32\borland\bc54\libeng.lib
   \matlabr12\extern\lib\win32\borland\bc54\libmat.lib
   \matlabr12\extern\lib\win32\borland\bc54\libmx.lib

2) copy the following Matlab dlls into the program directory:
   c:\matlabr12\bin\win32\libeng.dll
   c:\matlabr12\bin\win32\libmat.dll
   c:\matlabr12\bin\win32\libmx.dll

3) Add the library files to the project
