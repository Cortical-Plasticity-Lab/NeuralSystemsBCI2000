About

BCPy2000 is a software project originated by researchers at the Max Planck Institute for Biological Cybernetics in 2007.

It provides a platform for rapid, flexible development of experimental Brain-Computer Interface systems based on the BCI2000. From the developer's point of view, the implementation is carried out in python, taking advantage of various high-level packages: VisionEgg for stimulus presentation, NumPy and SciPy for signal processing and classification, and IPython for interactive debugging.

Being a BCI2000 system, it is modular. This means you can choose to use python for the Application module, and/or the Signal Processing module, and/or the Signal Source module, entirely as you please, and use other pre-existing BCI2000 modules for the remainder of the system.

BCPy2000 is aimed at developers. Currently, our emphasis is on getting you to the stage where you can implement a BCI2000 module based entirely on your own python code. We provide a lot of infrastructure to allow you to do so quickly, but the details of exactly how you classify your signals, or how you arrange your stimuli in space and time, are going to be left to you. We do provide a demo implementation which makes use of some optional higher-level tools that you may also find useful. However, the optional tools are currently only patchily documented. We're sure your own implementations will be better anyway.

To learn more, you may wish to take a look at how the system is structured.
