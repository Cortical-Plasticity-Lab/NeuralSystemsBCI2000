Python Packages

The table below shows download information on BCPy2000's dependencies. Since we are dependent on Windows, the version numbers are direct links to the win32 binary installers for Python 2.4 and Python 2.5. If you do not already have either version, then we recommend you stick to 2.5. Once downloaded, each package usually only takes a few seconds to install. Make sure, however, that you install them in the order listed.

If you want to install any of these packages from their sources (for example, to install VisionEgg 1.1.1 under Python 2.4) then pure python packages (such as pyreadline and ipython) can be installed with the command 

setup.py install

The rest require compilation of C or C++ sources, which requires either the official Microsoft Visual Studio 7.3, or, if you are lucky, MinGW 5.1. After ensuring that MinGW is on the path, for example by saying 

set PATH=C:\MinGW\bin;%PATH%

you can sometimes compile and install a package successfully with 

setup.py build -c mingw32 install
