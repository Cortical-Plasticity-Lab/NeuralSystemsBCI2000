VisionEgg

The Vision Egg is a high-level python framework written by Andrew Straw. It is aimed at experimental psychologists and vision scientists, and its main purpose is the presentation and control of visual stimuli.

BCPy2000 visual stimulus objects are created by instantiating VisionEgg classes, so you may wish to familiarize yourself with it at the official site http://visionegg.org and bookmark the reference manual.

The BCPy2000 Application module automatically handles the updating and redrawing of stimuli. All you need to do is:

  1. Optionally, during your Preflight hook, set any VisionEgg.config parameters you want in order to specify window size and behaviour before VisionEgg opens.
  2. During your Initialize hook, instantiate the stimuli and register them with calls to self.stimulus. For example:

    from VisionEgg.Text import Text
    ...
    t = Text('Hello World', position=(100,100)))
    self.stimulus('blah', t)

  3. During your Process, Transition, or Frame hooks, update the stimulus parameters to create the behaviour you want. For example:

    self.stimuli['blah'].parameters.color=(1,0,0)

    will turn the stimulus you have registered as 'blah' red.

Once the VisionEgg window has been opened (following "Set Config") the current VisionEgg.Core.Screen object is available at self.screen and the current VisionEgg.Core.Viewport is at self.viewport, but it is not usually necessary to access these directly. One common exception might be in order to change self.screen.parameters.bgcolor.
